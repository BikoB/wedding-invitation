# Свадебное приглашение с опросником

## Что реализовано
- При входе сначала запрашиваются имя и фамилия.
- После входа можно выбрать: `Приду на свадьбу` или `Не приду`.
- Ниже показываются 2 списка:
  - кто нажал `Приду`
  - кто нажал `Не приду`
- Ответы хранятся на сервере в `data/responses.json`.

## Запуск
1. Установите [Node.js](https://nodejs.org/).
2. В папке проекта выполните:
   ```bash
   npm install
   npm start
   ```
3. Откройте в браузере:
   `http://localhost:3000`

## Деплой на сервер
- Загрузите все файлы проекта на сервер.
- Установите зависимости: `npm install`.
- Запустите приложение: `npm start`.
- Для продакшена лучше запускать через `pm2` или systemd + reverse proxy (Nginx).
